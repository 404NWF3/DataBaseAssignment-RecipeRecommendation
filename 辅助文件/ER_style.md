
## 3. E-R 图符号 (E-R Diagram Components)

| 图形符号                           | 代表含义 (Meaning)                |
| :----------------------------- | :---------------------------- |
| **矩形 (Rectangles)**            | 实体集 (Entity sets)             |
| **椭圆 (Ellipses)**              | 属性 (Attributes)               |
| **菱形 (Diamonds)**              | 联系集 (Relationship sets)       |
| **线 (Lines)**                  | 连接属性与实体，或实体与联系                |
| **双椭圆 (Double ellipses)**      | 多值属性 (Multivalued attributes) |
| **虚线椭圆 (Dashed ellipses)**     | 派生属性 (Derived attributes)     |
| **下划线 (Underlined attribute)** | 主键 (Primary key)              |
